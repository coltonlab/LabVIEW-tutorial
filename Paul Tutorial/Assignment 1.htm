Programming Assignment 1: Unit Converter

Problem:

You need to create a unit converter program capable of converting miles to kilometers, feet to meters, and inches to centimeters.  The conversion factors are:

1mi = 1.609344km
1ft = 0.3048m
1in = 2.54cm

Requirements:

To simplify this assignment (we wouldn't want the first one to be too difficult!), you are allowed the following on  your front panel:

1. 3 numerical inputs
2. 3 numerical outputs
3. 3 boolean buttons labeled "convert"
4. 1 stop button

NOTE: It is advised to use labels for most if not all objects placed on the front panel.  Not only do you want to create a working program, but you want to create an interface that is easy for anyone to use and understand.

In order to establish a real-time interface in which you can repeatedly make conversions of all three types, you are recommended to utilize a while loop and event structures.  This means that when your program is run, the following must be true:

1. It continues running until a stop button is pressed.
2. The conversions occur immediately when their corresponding boolean buttons are pressed.
3. It can perform repeated conversions of all types.

 

Sample Solution

NOTE: Since there are so many ways in which a unit converter could be created, the sample solution is far from the best solution.  Though easy to use, this particular interface can seem bulky and slow.  Even when you have a working program that fits all requirements, you are encouraged to try and improve upon it.  For example, a much better program would utilize 1 numerical input, 1 numerical output, and boolean controls to select which conversion to perform.  So, as you learn more about LabView and become more familiarized with it, revisit your program to see if you can improve it even more.
